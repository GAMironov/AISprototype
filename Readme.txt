Файлы с конфигурационными данными находятся:
	1) Для проекта PeopleDB_Server1 в папке App_Data. В файле прописано подключение к Базе.
	2) Для проекта PeopleDB_Client2 в папке с .exe файлом приложения. В файле прописан адрес сервера.

PeopleDBinit.sql - файл для создания БД, хранимых процедур и первоначального заполнения базы данными.

Версия .NET Framework 4.6.1.
Среда разработки: Microsoft Visual Studio Professional 2017.
